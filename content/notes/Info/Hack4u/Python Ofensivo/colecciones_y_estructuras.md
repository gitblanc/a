## Listas

En esta clase, nos sumergiremos en profundidad en uno de los tipos de datos m√°s vers√°tiles y utilizados en Python.

Las listas son estructuras de datos que nos permiten almacenar secuencias ordenadas de elementos. Son mutables, lo que significa que podemos modificarlas despu√©s de su creaci√≥n, y son din√°micas, permiti√©ndonos a√±adir o quitar elementos de ellas.

**Caracter√≠sticas de las Listas**

Vamos a explorar las caracter√≠sticas clave de las listas en Python, que incluyen su capacidad para:

- Almacenar datos heterog√©neos, es decir, pueden contener elementos de diferentes tipos (enteros, cadenas, flotantes y m√°s) dentro de una misma lista.
- Ser indexadas y cortadas, lo que permite acceder a elementos espec√≠ficos de la lista directamente a trav√©s de su √≠ndice.
- Ser anidadas, es decir, una lista puede contener otras listas como elementos, lo que permite crear estructuras de datos complejas como matrices.

**Operaciones con Listas**

Tambi√©n cubriremos las operaciones fundamentales que se pueden realizar con listas, como:

- A√±adir elementos con m√©todos como ‚Äò**append()**‚Äò y ‚Äò**extend()**‚Äò.
- Eliminar elementos con m√©todos como ‚Äò**remove()**‚Äò y ‚Äò**pop()**‚Äò.
- Ordenar las listas con el m√©todo ‚Äò**sort()**‚Äò o la funci√≥n incorporada ‚Äò**sorted()**‚Äò.
- Invertir los elementos con el m√©todo ‚Äò**reverse()**‚Äò o la sintaxis de corte ‚Äò**[::-1]**‚Äò.
- Comprender las comprensiones de listas, una forma ‚Äúpythonica‚Äù de crear y manipular listas de manera concisa y eficiente.

**M√©todos de Listas**

Profundizaremos en la rica gama de m√©todos que Python ofrece para trabajar con listas y c√≥mo estos m√©todos pueden ser utilizados para manipular listas de acuerdo a nuestras necesidades.

**Buenas Pr√°cticas**

Discutiremos las mejores pr√°cticas en el manejo de listas, incluyendo c√≥mo y cu√°ndo usar listas en comparaci√≥n con otros tipos de colecciones en Python, como tuplas, conjuntos y diccionarios.

Al final de esta clase, tendr√°s un conocimiento profundo de las listas en Python y estar√°s equipado con las t√©cnicas para manejarlas eficazmente en tus programas. Con esta base s√≥lida, podr√°s manipular colecciones de datos con confianza y aplicar esta habilidad central en tareas como la manipulaci√≥n de datos, la automatizaci√≥n y el desarrollo de algoritmos.

## Tuplas

En esta clase, dedicaremos nuestro enfoque a las tuplas, una estructura de datos fundamental en Python que comparte algunas similitudes con las listas, pero se distingue por su inmutabilidad.

Las tuplas son colecciones ordenadas de elementos que no pueden modificarse una vez creadas. Esta caracter√≠stica las hace ideales para asegurar que ciertos datos permanezcan constantes a lo largo del ciclo de vida de un programa.

**Caracter√≠sticas de las Tuplas**

- **Inmutabilidad**: Una vez que se crea una tupla, no puedes cambiar, a√±adir o eliminar elementos. Esta inmutabilidad garantiza la integridad de los datos que desea mantener constantes.
- **Indexaci√≥n y Slicing**: Al igual que las listas, puedes acceder a los elementos de la tupla mediante √≠ndices y tambi√©n puedes realizar operaciones de slicing para obtener subsecuencias de la tupla.
- **Heterogeneidad**: Las tuplas pueden contener elementos de diferentes tipos, incluyendo otras tuplas, lo que las hace muy vers√°tiles.

**Operaciones con Tuplas**

Aunque no puedes modificar una tupla, hay varias operaciones que puedes realizar:

- **Empaquetado y Desempaquetado de Tuplas**: Las tuplas permiten asignar y desasignar sus elementos a m√∫ltiples variables de forma simult√°nea.
- **Concatenaci√≥n y Repetici√≥n**: Similar a las listas, puedes combinar tuplas usando el operador ‚Äò**+**‚Äò y repetir los elementos de una tupla un n√∫mero determinado de veces con el operador ‚Äò*****‚Äò.
- **M√©todos de B√∫squeda**: Puedes usar m√©todos como ‚Äò**index()**‚Äò para encontrar la posici√≥n de un elemento y ‚Äò**count()**‚Äò para contar cu√°ntas veces aparece un elemento en la tupla.

**Uso de Tuplas en Python**

- **Funciones y Asignaciones M√∫ltiples**: Las tuplas son muy √∫tiles cuando una funci√≥n necesita devolver m√∫ltiples valores o cuando se realizan asignaciones m√∫ltiples en una sola l√≠nea.
- **Estructuras de Datos Fijas**: Se usan para crear estructuras de datos que no deben cambiar, como los d√≠as de la semana o las coordenadas de un punto en el espacio.

**Buenas Pr√°cticas**

Abordaremos cu√°ndo es m√°s apropiado utilizar tuplas en lugar de listas y c√≥mo la elecci√≥n de una tupla sobre una lista puede afectar la claridad y la seguridad del c√≥digo.

Al concluir esta clase, tendr√°s un entendimiento claro de qu√© son las tuplas, c√≥mo y cu√°ndo utilizarlas en tus programas, y las pr√°cticas recomendadas para trabajar con este tipo de datos inmutable. Las tuplas son una herramienta poderosa en Python, y saber c√≥mo utilizarlas te permitir√° escribir c√≥digo m√°s seguro y eficiente.

```python
# Son inmutables

example2 = (1, "test", [1,2,3], 4, True, {'manzanas': 1, 'peras': 5}, 5) 
a, b, c, d = example # cada variable se asigna sola a cada valor de la tupla

```

## Conjuntos (Sets)

En esta clase, nos adentraremos en los conjuntos, conocidos en Python como ‚Äò**sets**‚Äò. Los conjuntos son una colecci√≥n de elementos sin orden y sin elementos repetidos, inspirados en la teor√≠a de conjuntos de las matem√°ticas. Son ideales para la gesti√≥n de colecciones de elementos √∫nicos y operaciones que requieren eliminar duplicados o realizar comparaciones de conjuntos.

**Caracter√≠sticas de los Conjuntos**

- **Unicidad**: Los conjuntos autom√°ticamente descartan elementos duplicados, lo que los hace perfectos para recolectar elementos √∫nicos.
- **Desordenados**: A diferencia de las listas y las tuplas, los conjuntos no mantienen los elementos en ning√∫n orden espec√≠fico.
- **Mutabilidad**: Los elementos de un conjunto pueden ser agregados o eliminados, pero los elementos mismos deben ser inmutables (por ejemplo, no puedes tener un conjunto de listas, ya que las listas se pueden modificar).

**Operaciones con Conjuntos**

Exploraremos las operaciones b√°sicas de conjuntos que Python facilita, como:

- **Adici√≥n y Eliminaci√≥n**: A√±adir elementos con ‚Äò**add()**‚Äò y eliminar elementos con ‚Äò**remove()**‚Äò o ‚Äò**discard()**‚Äò.
- **Operaciones de Conjuntos**: Realizar uniones, intersecciones, diferencias y diferencias sim√©tricas utilizando m√©todos o operadores respectivos.
- **Pruebas de Pertenencia**: Comprobar r√°pidamente si un elemento es miembro de un conjunto.
- **Inmutabilidad Opcional**: Usar el tipo ‚Äò**frozenset**‚Äò para crear conjuntos que no se pueden modificar despu√©s de su creaci√≥n.

**Uso de Conjuntos en Python**

- **Eliminaci√≥n de Duplicados**: Son √∫tiles cuando necesitas asegurarte de que una colecci√≥n no tenga elementos repetidos.
- **Relaciones entre Colecciones**: Facilitan la comprensi√≥n y el manejo de relaciones matem√°ticas entre colecciones, como subconjuntos y superconjuntos.
- **Rendimiento de B√∫squeda**: Proporcionan una b√∫squeda de elementos m√°s r√°pida que las listas o las tuplas, lo que es √∫til para grandes vol√∫menes de datos.

**Buenas Pr√°cticas**

Discutiremos cu√°ndo es beneficioso usar conjuntos en lugar de otras estructuras de datos y c√≥mo su uso puede influir en la eficiencia del programa.

Al final de esta clase, tendr√°s una comprensi√≥n completa de los conjuntos en Python y c√≥mo pueden ser utilizados para hacer tu c√≥digo m√°s eficiente y l√≥gico, aprovechando sus propiedades √∫nicas para manejar datos. Con este conocimiento, podr√°s implementar estructuras de datos complejas y operaciones que requieren l√≥gica de conjuntos.

```python
conjunto = {1,2,3}
conjunto2 = {1,3,5,6,7}
conjunto.discard(7); # si no existe no salta excepci√≥n como con .remove()

conjunto_final2 = conjunto.union(conjunto2); # [out] {1,2,3,5,6,7}

# .issubset() para comprobar si un conjunto es subconjunto de otro

```

## Diccionarios

En esta clase, nos centraremos en los diccionarios, una de las estructuras de datos m√°s poderosas y flexibles de Python. Los diccionarios en Python son colecciones desordenadas de pares clave-valor. A diferencia de las secuencias, que se indexan mediante un rango num√©rico, los diccionarios se indexan con claves √∫nicas, que pueden ser cualquier tipo inmutable, como cadenas o n√∫meros.

**Caracter√≠sticas de los Diccionarios**

- **Desordenados**: Los elementos en un diccionario no est√°n ordenados y no se accede a ellos mediante un √≠ndice num√©rico, sino a trav√©s de claves √∫nicas.
- **Din√°micos**: Se pueden agregar, modificar y eliminar pares clave-valor.
- **Claves √önicas**: Cada clave en un diccionario es √∫nica, lo que previene duplicaciones y sobrescrituras accidentales.
- **Valores Accesibles**: Los valores no necesitan ser √∫nicos y pueden ser de cualquier tipo de dato.

**Operaciones con Diccionarios**

Durante la clase, exploraremos c√≥mo realizar operaciones b√°sicas y avanzadas con diccionarios:

- **Agregar y Modificar**: C√≥mo agregar nuevos pares clave-valor y modificar valores existentes.
- **Eliminar**: C√≥mo eliminar pares clave-valor usando del o el m√©todo ‚Äò**pop()**‚Äò.
- **M√©todos de Diccionario**: Utilizar m√©todos como ‚Äò**keys()**‚Äò, ‚Äò**values()**‚Äò, y ‚Äò**items()**‚Äò para acceder a las claves, valores o ambos en forma de pares.
- **Comprensiones de Diccionarios**: Una forma elegante y concisa de construir diccionarios basados en secuencias o rangos.

**Uso de Diccionarios en Python**

- **Almacenamiento de Datos Estructurados**: Ideales para almacenar y organizar datos que est√°n relacionados de manera l√≥gica, como una base de datos en memoria.
- **B√∫squeda Eficiente**: Los diccionarios son altamente optimizados para recuperar valores cuando se conoce la clave, proporcionando tiempos de b√∫squeda muy r√°pidos.
- **Flexibilidad**: Pueden ser anidados, lo que significa que los valores dentro de un diccionario pueden ser otros diccionarios, listas o cualquier otro tipo de dato.

**Buenas Pr√°cticas**

Enfatizaremos las mejores pr√°cticas para trabajar con diccionarios, incluyendo la selecci√≥n de claves adecuadas y el manejo de errores comunes, como intentar acceder a claves que no existen.

Al final de esta clase, tendr√°s una comprensi√≥n completa de los diccionarios y estar√°s listo para utilizarlos para gestionar eficazmente los datos dentro de tus programas. Los diccionarios son una herramienta esencial en Python y saber c√≥mo utilizarlos te abrir√° la puerta a un nuevo nivel de programaci√≥n.

```python

	print(f"Para la clave {key} tenemos el valor {value}")

	print(elem)

for elem in diccionario.values():
	print(elem)
```
